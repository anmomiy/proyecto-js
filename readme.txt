PROYECTO CREADO PARA EL CURSO DE JAVASCRIPT POR ADRIANA MOMIY

El proyecto es una continuación de lo que había hecho en el curso de Desarrollo Web.

Tuve que rehacer la página de productos para crear cada card desde javascript. Para ello, se transformó la información de los productos al formato JSON y se convirtió en una api en Beeceptor.com, finalmente se trajo todo con fetch y se crearon los elementos. La Api creada en Beeceptor.com solo permite 50 requests por día así que, si no aparecen los productos o la consola da un error 404, es porque se excedió el número de requests por día. En caso de superar los 50 requests, usar https://quinpro2.free.beeceptor.com 
El sitio web te permite seleccionar productos, agregarlos al carrito y generar una orden de compra. También permite quitar productos del carrito. 

En cuanto a los formularios del footer y página de contacto, se usó la Api Fromspree. Toda la información ingresada en dichos formularios me llegará a mi correo personal.
